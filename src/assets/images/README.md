# Bilder-Verzeichnis

## Struktur

### `/profile`
Hier sollte Ihr Profilbild für den Hero-Bereich abgelegt werden.
- Empfohlene Dateigröße: < 500 KB
- Empfohlenes Format: JPG oder WebP
- Empfohlene Auflösung: 800x800 px oder größer (quadratisch)

### `/projects`
Hier können Screenshots oder Vorschaubilder Ihrer Projekte abgelegt werden.
- Empfohlene Dateigröße: < 300 KB pro Bild
- Empfohlenes Format: JPG oder WebP
- Empfohlenes Seitenverhältnis: 16:9

### `/bg`
Hier können Hintergrundbilder oder dekorative Grafiken abgelegt werden.
